Reparameterization Trick – Numerical Example

1. Assume Encoder Output

\[ \mu = 2 \] \[ \sigma = 0.5 \]

The encoder predicts a Gaussian distribution:

\[ q(z|x) = N(2, 0.5^2) \]

2. Sample Standard Noise

Instead of sampling from the Gaussian directly, we sample:

\[ \epsilon \sim N(0,1) \]

Example values:

εz = μ + σ ε
-11.5
02.0
12.5
23.0

3. Gradient Intuition

Now the sampled variable is computed as:

\[ z = \mu + \sigma \epsilon \]

So gradients exist:

\[ \frac{\partial z}{\partial \mu} = 1 \] \[ \frac{\partial z}{\partial \sigma} = \epsilon \]
The randomness is isolated in ε, which does not depend on network parameters. Therefore gradients can propagate through μ and σ.

4. Intuition

A Gaussian sample can always be expressed as:

\[ sample = location + scale \times noise \]

This simple idea is what makes Variational Autoencoders trainable.
