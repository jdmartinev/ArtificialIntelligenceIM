Variational Autoencoders

ELBO and Reparameterization Trick

1. Goal of a Generative Model

We want a model that can generate data similar to the training dataset.

Assume a latent variable model:

\[ p(x,z) = p(x|z)p(z) \]

The probability of a datapoint is:

\[ p(x) = \int p(x,z)dz \]

Training ideally maximizes:

\[ \log p(x) \]

2. Variational Inference

The posterior \(p(z|x)\) is usually intractable, so we approximate it with:

\[ q(z|x) \]

This is implemented as the encoder network.

3. Evidence Lower Bound (ELBO)

\[ \log p(x) = \log \int q(z|x) \frac{p(x,z)}{q(z|x)} dz \]

Using Jensen's inequality:

\[ \log p(x) \ge E_{q(z|x)}\left[ \log \frac{p(x,z)}{q(z|x)} \right] \]
This expression is called the Evidence Lower Bound (ELBO).

4. ELBO Decomposition

\[ ELBO = E_{q(z|x)}[\log p(x|z)] - KL(q(z|x)||p(z)) \]

The first term encourages good reconstruction. The second term regularizes the latent space.

5. Reparameterization Trick

If:

\[ q(z|x) = N(\mu,\sigma^2) \]

Instead of sampling directly:

\[ z \sim N(\mu,\sigma^2) \]

We sample:

\[ \epsilon \sim N(0,1) \]

and compute:

\[ z = \mu + \sigma \epsilon \]
This makes the sampling operation differentiable and allows gradients to flow through the network.
